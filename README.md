# 🔐 Formulário de Login

Projeto desenvolvido com HTML e CSS criando um formulário de login simples, com foco em design moderno e uso de gradiente.

---

## 🛠️ Tecnologias utilizadas
- HTML
- CSS

---

## 🎯 Objetivo
Praticar a criação de formulários estilizados, trabalhando com layout, cores e organização de elementos.

- Alinhamento e organização de elementos  
- Criação de layout simples e funcional
